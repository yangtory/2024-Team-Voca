## 1. 개발환경
- 언어 : Javascript, Pug, HTML5, CSS
- 서버 : NodeJS, Express
- 프레임워크 : Express, Sequelize
- DB : MySQL
- IDE : STS3
- API, 라이브러리 : Google Cloud API(Translate API), Multer, Session, UUID
- 협업 : Github, Notion
<br/>
